Инструкция:

1. Запустить программу 'check.bat';
2. Найти строки вида: 'Alternative name "@device_pnp_\\?\usb#vid_046d&pid_0825&mi_00#7&25d1bbbb&0&0000#{65e8773d-8f56-11d0-a3b9-00a0c9223196}\global"', где:
  '@device_pnp_\\?\usb#vid_046d&pid_0825&mi_00#' - префикс идентификаторов камер,
  '7&25d1bbbb&0' - идентификатор камеры,
  '&0000#{65e8773d-8f56-11d0-a3b9-00a0c9223196}\global' - суффикс идентификаторов камер;
2. Префикс и суффикс идентификаторов камер необходимо скопировать в файл 'config.conf':
  prefix=...
  suffix=...
3. Идентификаторы камер необходимо скопировать в файл 'cameras.conf' в порядке расположения камер;
4. Запустить программу 'capture.bat'.

P.S.: Для выполнения съемки в случайном порядке можно использовать программы 'capture_old.bat' и 'capture_sync.bat':
  capture_old.bat "USB Video Device" 6, где:
  capture.bat - имя программы (м.б. capture_sync.bat для синхронного выполнения),
  "USB Video Device" - имя камеры (см. программу 'check.bat'),
  6 - количество камер.
